# Guiding the sheep
 Guiding The Sheep is a libgdx-based game that use box2d to physics and implements 
pocketsphinx to voice recognition in the Android module, its incomplete so i decided
to uploaded to github and make it public, if someone needs to see a Libgdx-Pocketspnix
integration can use this project as a guide.

# Spanish comment
Aplicacion Android, programado en java con el framework Libgdx, fue semifinalista del concurso "Potenciate Videojuegos 2018" del GCBA, se trataba de controlar al personaje con comandos de voz usando la libreria Pocketsphinx para permitir el reconocimiento off-line. El juego en un principio se llamaba "Entrenando a Bingo" en honor a mi perro perdido. link de la noticia:
https://www.buenosaires.gob.ar/vicejefatura/bajoven/noticias/90-semifinlistas-de-potenciate-videojuegos-q
